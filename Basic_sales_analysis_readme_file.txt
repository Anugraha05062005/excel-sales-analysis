Basic Sales Analysis Using Excel

Project Description
This is a beginner-level data analysis project created using Microsoft Excel.
The project analyzes sales data to identify total sales, top-performing categories,
and product-level insights.

Tools Used
Microsoft Excel

Key Analysis Performed
- Data cleaning and formatting
- Total sales calculation
- Sales by category using SUMIF
- Visualizations using column and pie charts
- Business insights generation

Outcome
Electronics category generated the highest revenue, while stationery items
had higher sales volume but lower revenue contribution.
